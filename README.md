# 日日好日商務中心 — 官方網站

桃園中壢共享辦公室・公司借址登記・一站式創業服務

## 網站結構

```
/                         ← 根目錄（直接部署到 GitHub Pages）
├── index.html            首頁
├── services.html         我們的服務
├── space.html            辦公環境
├── news.html             最新消息（文章列表）
├── contact.html          聯絡我們
├── style.css             共用樣式
├── .nojekyll             停用 Jekyll（GitHub Pages 必要）
├── 404.html              自訂 404 頁面
└── news/                 創業知識文章
    ├── 01-中壢公司登記地址怎麼選.html
    ├── 02-剛創業沒有地址怎麼辦.html
    ├── 03-借址登記-共享辦公室-傳統辦公室怎麼選.html
    └── 04-借址登記會不會有問題.html
```

## 部署說明（GitHub Pages）

1. 建立 GitHub repository
2. 將本資料夾所有檔案推送到 `main` branch
3. Repository Settings → Pages → Source 選擇 `main` branch / `root`
4. 等待約 1 分鐘，網站即上線

> 所有圖片已 base64 嵌入，無需額外 assets 資料夾
